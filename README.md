# Tuberculosis-Detection-Using-Deep-Learning
An end-to-end deep learning system for Tuberculosis detection from chest X-ray images using PyTorch. Trained Custom CNN, ResNet50, and EfficientNetB0 with transfer learning on 3008 images. Best model (EfficientNetB0) achieved 98.67% accuracy and 100% AUC. Includes a Streamlit dashboard for real-time predictions with Grad-CAM visualization.
